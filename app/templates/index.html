 SQL Query Help

                
            

            

                
This tool runs read-only queries. Only SELECT, PRAGMA, EXPLAIN, and WITH are allowed.


                
Basic Queries

                

                    
QueryDescription

                    
                        
SELECT * FROM table_nameGet all rows and columns

                        
SELECT col1, col2 FROM tGet specific columns

                        
SELECT DISTINCT col FROM tUnique values only

                        
SELECT COUNT(*) FROM tCount total rows

                        
SELECT * FROM t LIMIT 10Limit result rows

                        
SELECT * FROM t LIMIT 10 OFFSET 20Skip first 20 rows, return next 10

                    
                


                
WHERE — Filtering Rows

                

                    
OperatorExampleMeaning

                    
                        
=WHERE name = 'Alice'Equals

                        
!= or <>WHERE age != 25Not equal

                        
< > <= >=WHERE age >= 18Comparison

                        
BETWEENWHERE id BETWEEN 10 AND 50Range (inclusive)

                        
INWHERE status IN ('active','pending')Match any in list

                        
IS NULLWHERE email IS NULLValue is NULL

                        
IS NOT NULLWHERE email IS NOT NULLValue is not NULL

                        
AND / ORWHERE age > 18 AND status = 'active'Combine conditions

                        
NOTWHERE NOT status = 'deleted'Negate condition

                    
                


                
LIKE — Wildcards & Pattern Matching

                

                    
PatternExampleMeaning

                    
                        
%WHERE name LIKE '%smith%'Matches any sequence of characters

                        
_WHERE code LIKE 'A_C'Matches exactly one character

                        
LIKE 'abc%'WHERE name LIKE 'John%'Starts with "John"

                        
LIKE '%xyz'WHERE file LIKE '%.pdf'Ends with ".pdf"

                        
GLOBWHERE path GLOB '*/logs/*'Case-sensitive, uses * and ?

                    
                


                
ORDER BY & GROUP BY

                

                    
ClauseExampleDescription

                    
                        
ORDER BYORDER BY created_at DESCSort results (ASC or DESC)

                        
GROUP BYSELECT type, COUNT(*) FROM t GROUP BY typeAggregate by groups

                        
HAVINGGROUP BY type HAVING COUNT(*) > 5Filter groups

                    
                


                
JOIN — Combining Tables

                

                    
TypeExampleDescription

                    
                        
INNER JOINSELECT * FROM a JOIN b ON a.id = b.a_idOnly matching rows from both tables

                        
LEFT JOINSELECT * FROM a LEFT JOIN b ON a.id = b.a_idAll rows from left + matches from right

                        
CROSS JOINSELECT * FROM a CROSS JOIN bAll combinations (cartesian product)

                    
                


                
Aggregate Functions

                

                    
FunctionDescription

                    
                        
COUNT(*)Number of rows

                        
SUM(col)Sum of values

                        
AVG(col)Average value

                        
MIN(col) / MAX(col)Minimum / maximum value

                        
GROUP_CONCAT(col, ',')Concatenate values into a string

                    
                


                
Useful SQLite Functions

                

                    
FunctionDescription

                    
                        
LENGTH(col)String length

                        
UPPER(col) / LOWER(col)Case conversion

                        
SUBSTR(col, start, len)Substring extraction

                        
REPLACE(col, 'old', 'new')String replacement (in results only)

                        
COALESCE(a, b, c)First non-NULL value

                        
TYPEOF(col)Data type of a value

                        
HEX(col)Hex representation of BLOB/text

                        
datetime(col, 'unixepoch')Convert Unix timestamp to readable date

                        
CAST(col AS INTEGER)Type conversion

                    
                


                
PRAGMA — Database Info

                

                    
CommandDescription

                    
                        
PRAGMA table_info('table')Column names and types

                        
PRAGMA table_listList all tables

                        
PRAGMA index_list('table')Indexes on a table

                        
PRAGMA database_listAttached databases

                        
PRAGMA integrity_checkCheck database integrity

                        
PRAGMA journal_modeCurrent journal mode

                    
                


                
Subqueries

                

                    
ExampleDescription

                    
                        
SELECT * FROM t WHERE id IN (SELECT id FROM t2)Filter with a subquery

                        
SELECT *, (SELECT COUNT(*) FROM t2 WHERE t2.ref = t.id) AS cnt FROM tScalar subquery

                        
WITH cte AS (SELECT ...) SELECT * FROM cteCommon Table Expression (CTE)

                    
                


                
Tips

                
    
                    
  • Use double quotes for table/column names with spaces: "My Table"
    • 
                    
  • Use single quotes for string values: 'hello'
    • 
                    
  • Queries without LIMIT are auto-capped at 500 rows for safety
    • 
                    
  • Ctrl+Enter runs the query from the editor
    • 
                    
  • Use the table/column dropdowns above the editor to insert names
    • 
                    
  • Autocomplete suggestions appear as you type
